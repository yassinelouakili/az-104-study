# AZ-104 Study Journey

Repositorio de estudio para la certificacion Microsoft AZ-104: Azure Administrator.

Estado: `En Progreso`

## Objetivo

Este repositorio documenta el proceso de preparacion para la certificacion Microsoft AZ-104: Azure Administrator, incluyendo teoria, laboratorios practicos y simulacros realizados durante el estudio.

Cada fase combina tres fuentes:
1. **Teoría y ejercicios cortos** de Microsoft Learn.
2. **Lab completo** del repositorio oficial de Microsoft Learn.
3. **Documentación** en este repo (`notes.md` + práctica en `labs/`).
